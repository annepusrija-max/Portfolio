Personal Portfolio Website

Objective

The objective of this project is to design and develop a personal portfolio website to showcase my skills, projects, internships, and achievements as part of the Oasis Infobyte Web Development Internship (OIBSIP).

Tools & Technologies Used

- HTML
- CSS
- GitHub
- GitHub Pages

Project Description

This project is a responsive personal portfolio website developed using HTML and CSS. The portfolio highlights my technical skills, academic projects, internships, hackathon participation, leadership activities, and contact information.

The website includes the following sections:

- About Me
- Skills
- Projects
- Internships
- Hackathons
- Leadership & Activities
- Languages
- Education
- Contact

Deployment

The website is hosted using GitHub Pages.

Live Portfolio Link:
https://smruti-1511.github.io/OIBSIP_WebDevelopment_Task1/

Outcome

This project helped me strengthen my understanding of web development fundamentals and improve my ability to present my work professionally through a personal portfolio website.
